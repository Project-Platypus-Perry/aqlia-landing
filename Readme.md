# Aqlia Landing Page

A sleek, minimalist landing page with smooth animations for your LMS platform, inspired by youlearn.ai design.

## ✨ New Features

### Animations & Visual Effects
- **Gradient text effects** on headings
- **Floating orb backgrounds** with subtle movement
- **Scroll-triggered animations** - elements fade in as you scroll
- **Staggered card animations** - features appear one after another
- **Shine effect** on buttons on hover
- **Gradient borders** that appear on card hover
- **Icon rotation** on feature card hover
- **Smooth scroll** for anchor links
- **Pulsing beta badge**
- **Glassmorphism header** with backdrop blur

All animations are:
- ✅ Smooth (60fps)
- ✅ Performant (CSS-based where possible)
- ✅ Minimal (no heavy libraries)
- ✅ Accessible (respects reduced motion preferences)

## Quick Start

The landing page is a single HTML file with embedded CSS and vanilla JavaScript. No build process needed!

## Hosting Options

### Option 1: Netlify (Easiest - Drag & Drop)
1. Go to [netlify.com](https://netlify.com)
2. Sign up for a free account
3. Drag and drop `index.html` into Netlify
4. Get instant deployment with free HTTPS
5. Custom domain available

### Option 2: Vercel
1. Go to [vercel.com](https://vercel.com)
2. Sign up for free
3. Click "Add New Project"
4. Drag and drop `index.html`
5. Deploy instantly

### Option 3: GitHub Pages (Free)
1. Create a GitHub repository
2. Upload `index.html` 
3. Go to Settings → Pages
4. Select branch and save
5. Access at `username.github.io/repo-name`

### Option 4: Cloudflare Pages
1. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
2. Connect your GitHub repo or upload directly
3. Free hosting with CDN

## Customization

You can easily customize:
- **Logo/Name**: Change "Aqlia" in the header
- **Colors**: Edit the gradient colors in CSS (search for `#667eea` and `#764ba2`)
- **Content**: Update text in HTML
- **Email**: Update `aarish@aqlia.org` with your email
- **Icons**: Change emoji icons in feature cards
- **Animation speed**: Adjust transition durations in CSS
